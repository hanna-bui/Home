Knitto Documentation

Introduction

Knitto is a web-based knitting pattern parser that transforms your written knitting instructions into organized spreadsheets and visual charts. This documentation will guide you through the simple process of using the app, customizing patterns, and taking advantage of its advanced features.

Getting Started

Pattern Input

To start, simply visit the app's website and enter the name of your pattern along with the written instructions into the provided textarea. Ensure that your instructions follow the specified format:

  • Single Row: "Row [number]: [stitch][number], [stitch][number], ...".
  • Repeating Rows: "Rows [first row] to [last row]: repeat rows [first row to repeat from]-[last row to repeat from]".
  • Repeating Stitches: "Row [number]: ..., *[stitch][number], ..., repeat from * to last [number] sts, ...".

Make sure to increment the row number consistently, and use the correct format for repeating stitches and rows.

Customization

Once uploaded, you can customize your pattern further, adjusting stitches, rows, and repetitions as needed.

Basic Usage

Visualizing Patterns

Your pattern will be visualized on an Excel or CSV spreadsheet, and a chart will be generated on the web for your convenience. An option is available for users to keep track of their current row while working on their knitting projects.

Advanced Features

Repeating Rows

Take advantage of the advanced feature to use repeating rows multiple times throughout your pattern, providing flexibility and efficiency in your knitting projects.

File Format

Download

Download your visualized knitting pattern in CSV format for easy sharing or printing.

FAQs

Missing Row Numbers

Ensure that your written instructions include an incrementing row number for each row to maintain accurate pattern visualization. Missing row numbers may result in incorrect output.

Stitch Abbreviations

  • Abbreviations of knit and purl must be followed by a number.
  • When using the word version of knit and purl, the program will knit/purl the same amount of stitches as the previous row.
  • Other stitches don't require a number after the stitch abbreviation.

Repeating Stitches

  • Repeating stitches must be written in a single row format and can only be used once per row.
  • The format is "Row [number]: ..., *[stitch][number], ..., repeat from * to last [number] sts, ..." or "Row [number]: ..., *[stitch][number], ..., repeat from * to last st, ...".
